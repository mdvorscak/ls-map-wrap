A wrapper around localstorage, so it can be used as a Map

## Getting Started

Install it via npm:

```shell
npm install ls-map-wrap
```

And include in your project:

```javascript
// Using ESM
import lsMap from 'ls-map-wrap';
// Or using CJS
const lsMap = require('ls-map-wrap');
```
